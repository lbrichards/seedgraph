# Git Ignore Configuration

## What's Ignored (Not Tracked)

### SeedGraph Data & Artifacts
- `checkpoints/*` - All checkpoint JSONL files (can be 100MB+)
- `data/*` - All data files
- `*.jsonl` - All JSONL files (checkpoints, exports, etc.)
- `models/` - Downloaded model weights (if stored locally)
- `.cache/` - Cache directory

### Large Binary Files
- `*.bin` - Binary model files
- `*.safetensors` - SafeTensors model format
- `*.onnx` - ONNX model files
- `*.pb` - Protocol buffer files

### Python Standard
- `__pycache__/` - Python bytecode cache
- `*.pyc`, `*.pyo`, `*.pyd` - Compiled Python
- `*.egg-info/` - Package metadata
- `.pytest_cache/` - Pytest cache
- `dist/`, `build/` - Distribution/build artifacts
- `.venv/`, `venv/`, `env/` - Virtual environments
- `.coverage*` - Coverage reports

### Temporary Files
- `run_*.json` - Temporary run files
- `*.tmp` - Temporary files
- `*.log` - Log files

## What's Tracked (Version Controlled)

### Source Code
- `src/seedgraph/**/*.py` - All Python source files
- `tests/**/*.py` - All test files

### Configuration
- `pyproject.toml` - Poetry configuration
- `poetry.lock` - Dependency lock file (for reproducibility)
- `Makefile` - Build automation

### Documentation
- `README.md` - Main documentation
- `QUICKSTART.md` - Quick start guide
- `IMPLEMENTATION_SUMMARY.md` - Technical documentation
- `SUCCESS_VERIFICATION.md` - Verification report
- `LICENSE` - License file

### Directory Structure
- `checkpoints/.gitkeep` - Keeps checkpoints/ in git
- `data/.gitkeep` - Keeps data/ in git

## Verification

```bash
# Check if a file is ignored
git check-ignore -v <filename>

# See what would be added
git add -n <directory>

# Verify checkpoint files are ignored
git check-ignore checkpoints/*.jsonl
```

## Notes

- Large checkpoint files (100MB+) are never committed to git
- Empty directories (`checkpoints/`, `data/`) are preserved via `.gitkeep`
- Model weights are stored in HuggingFace cache (`~/.cache/huggingface/`)
- Poetry lock file IS tracked for reproducible builds
